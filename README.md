<h1>Microsoft Azure Resource Groups – How To Create And Assign Tags Via The Azure Portal</h1>


<h2 style="font-family: Arial, sans-serif; font-size: 20px; font-weight: bold; margin-top: 24px; margin-bottom: 12px;">
⏹️ Description</h2>

<p style="font-family: Georgia, serif; font-size: 16px; margin-top: 12px; margin-bottom: 12px;">
In this project, you will be working within the Microsoft Azure portal to apply governance practices by organizing resources in a consistent and identifiable manner using resource groups and tags. You will create a new Azure resource group and assign a governance tag that identifies a cost center in alignment with Azure governance best practices. You access the Azure portal, navigate to Resource groups, and initiate the creation process by defining the resource group’s basic configuration, including subscription, name, and region. You then configure a tag by specifying a name and value on the Tags tab, review the configuration for accuracy, and complete the deployment by creating the resource group. The resource group is successfully created and tagged, making it visible after refreshing the Resource groups page, and provides a structured method to categorize and manage resources for governance, identification, and administrative purposes.
